# 读物资源 / Reading Materials

个人每日阅读输入，按附录 C 的五阶段学习路线组织。

## 目录结构

```
reading/
├── foundation/          # 阶段 0：基础巩固
│                        #   Ahlfors, Axler, Bhatia, Artin, Rotman
├── analytic-nt/         # 阶段 1：解析数论
│                        #   Edwards, Apostol, Ivić, Riemann 1859 原文
├── algebraic-nt/        # 阶段 2：代数数论
│                        #   Marcus, Neukirch, Stewart & Tall
├── automorphic-forms/   # 阶段 3：自守形式与 Langlands 纲领
│                        #   Bump, Gelbart, Serre, Langlands 致 Weil 信
├── spectral-ncg/        # 阶段 4：谱理论与非交换几何
│                        #   Connes, Sarnak, Chavel, Mehta
└── frontier/            # 阶段 5：前沿论文
                         #   Guth-Maynard, Scholze, Fargues, Keating-Snaith
```

## 使用方式

- 每个目录放对应阶段的 PDF、论文、笔记
- 每日阅读后可在对应目录中记录笔记
- 交叉主题的读物放在最相关的阶段，或创建子目录
